# Finance API

API REST para gerenciamento financeiro pessoal desenvolvida com Java e Spring Boot.

## Tecnologias

- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Lombok

## Funcionalidades

- Cadastro de usuários
- Cadastro de receitas
- Cadastro de despesas
- Dashboard financeiro
- Autenticação JWT (em desenvolvimento)

## Banco de Dados

PostgreSQL

## Como executar

```bash
mvn spring-boot:run
